## Featured Projects

### MindShelf
**Bookmark manager with web + iOS app, both in production.**

- Built a cross-platform product experience for saving and organizing bookmarks
- iOS app developed with **Swift, SwiftUI, StoreKit 2, SwiftData**
- Web app developed with **React + Vite**
- Cross-device sync powered by **Supabase**
- Metadata auto-fetch integrated with **Microlink** and **Noembed**
- Includes free tier limits, paywall flow, subscriptions, and restore purchases

**Live:** [mindshelf-web.pages.dev](https://mindshelf-web.pages.dev)  
**Tech:** Swift · SwiftUI · StoreKit 2 · SwiftData · React · Vite · Supabase · Cloudflare Pages

- Built a modern business website focused on product presentation and conversion
- Structured product and service content for a real commercial use case
- Focused on clean UI, trust-building layout, and clear communication
- Designed to support business visibility and client acquisition

**Live:** [softmendil.com](https://softmendil.com)
