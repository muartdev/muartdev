## Featured Projects

### MindShelf  
Bookmark manager with a web app and iOS companion app, both in production.

- iOS app built with **Swift, SwiftUI, StoreKit 2, SwiftData**
- Cross-device sync with **Supabase**
- Web app built with **React + Vite**
- Metadata auto-fetch with **Microlink** and **Noembed**
- Free tier limits, paywall, restore purchases and subscription flow
- Live: [mindshelf-web.pages.dev](https://mindshelf-web.pages.dev)

**Tech:** Swift · SwiftUI · StoreKit 2 · SwiftData · React · Vite · Supabase · Cloudflare Pages
